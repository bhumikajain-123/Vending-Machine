# Vending Machine Management System

## Description
This project is a Laravel-based vending machine system that allows managing products, inventory, and vending operations through a web interface.

## Features
- Add / Edit / Delete products
- Upload product images
- Admin dashboard
- User authentication
- Inventory tracking

## Tech Stack
- Laravel (PHP)
- MySQL
- HTML / CSS / JS
- Bootstrap

## Installation
1. Clone repo
2. Run composer install
3. Setup .env file
4. Run php artisan migrate
5. Run php artisan serve
